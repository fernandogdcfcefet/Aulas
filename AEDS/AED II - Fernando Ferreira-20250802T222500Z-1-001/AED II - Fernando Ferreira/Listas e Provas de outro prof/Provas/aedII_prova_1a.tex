\section*{Questőes}
\begin{enumerate}
	
	\item (1,0 pontos) Desenvolva um \textbf{programa} que solicite ao usuário a quantidade de alunos de uma turma e aloca, de forma dinâmica, um vetor de notas (números reais). Depois de ler as notas, imprime a média aritmética. 
	
	\item (2,0 pontos) Determine o que a seguinte funçăo recursiva calcula. Escreva uma funçăo iterativa para atingir o mesmo objetivo.
	
\begin{lstlisting}
int func (int n)
{
  if (n <= 0)
    return 0;
  return(n + func(n-2));
}
\end{lstlisting}	

\item (2,0 pontos) Crie um programa que receba um vetor de números reais com 100 elementos.
Escreva uma funçăo recursiva que inverta ordem dos elementos presentes no vetor.
	
\item (5,0) Dada a sequęncia de números: [3 5 1 4 2] ordene em ordem crescente utilizando os algoritmos aprendidos em sala (Bolha, Seleçăo, Inserçăo, MergeSort e QuickSort). Para cada algoritmo vocę deverá apresentar as seguintes informaçőes:

\begin{enumerate}
	\item sequęncia dos números a cada passo;
	\item número de comparaçőes realizadas;
	\item número de trocas realizadas.
\end{enumerate}
	
\end{enumerate}
