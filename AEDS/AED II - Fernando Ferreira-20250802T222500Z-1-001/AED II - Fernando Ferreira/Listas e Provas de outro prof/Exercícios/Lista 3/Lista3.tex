\section*{Filas}

\begin{enumerate}
\item Implemente uma fila para gerenciar dados do tipo \textbf{Cliente}, onde Cliente é uma ESTRUTURA com Nome e Idade. Implemente uma para a versão usando arranjos e outra para a versão usando ponteiros.

\item Implemente um procedimento CLONAR para a Fila, que retorna uma Fila exatamente igual à variável sobre a qual o método foi invocado. Implemente um para a versão usando arranjos e outro para a versão usando ponteiros.

\item Implemente uma função que receba como parâmetro duas Filas e retorne uma terceira contendo a concatenação das duas. Implemente uma para a versão usando arranjos e outra para a versão usando ponteiros.

\item Escreva um procedimento que inverta a ordem dos elementos na Fila. Implemente uma para a versão usando arranjos e outra para a versão usando ponteiros. \underline{DICA:} considera a utilização de estruturas auxiliares, como uma pilha.

\item Em muitas situações é necessário que exista uma prioridade na Fila, como por exemplo em Bancos onde pessoas idosas têm prioridade no atendimento. Altere a Fila criada no exercício 1 para que a inserção na Fila seja feita por prioridade de idade, ou seja, se o Cliente que tiver mais de 65 anos deve ter prioridade no atendimento. Implemente uma para a versão usando arranjos e outra para a versão usando ponteiros.

\noindent \underline{DICA:} Não é necessário criar a Fila toda novamente, apenas um novo procedimento de Enfileira por prioridade.

\end{enumerate}
