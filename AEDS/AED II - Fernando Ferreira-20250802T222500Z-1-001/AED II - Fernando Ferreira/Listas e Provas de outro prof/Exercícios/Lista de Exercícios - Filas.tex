	\begin{center}
		{\Large \bf Lista de Exercícios - Filas} \\
		%{\normalsize \bf em dupla}
	\end{center}
	

\begin{enumerate}
	
\item Implemente uma Fila para gerenciar dados do tipo \textbf{inteiro}. Sua TAD deve conter as seguintes operaçőes:

\begin{enumerate}
	\item Inicializar a Fila (FFVazia)
	\item Verificar se a Fila está vazia (Vazia)
	\item Enfileirar um item (Enfileirar)
	\item Desenfileirar um item (Desenfileirar)
	\item Retorna o tamanho da Fila (Tamanho)
	\item Imprime o conteúdo da Fila (Imprime)
\end{enumerate} 

\noindent Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Implemente um procedimento CLONAR para a Fila, que retorna uma Fila exatamente igual ŕ variável sobre a qual o método foi invocado. Seu procedimento deve usar apenas os métodos já implementados.

\item Implemente um procedimento que receba como parâmetro duas Filas e retorne uma terceira contendo a concatenaçăo das duas. Seu procedimento deve usar apenas os métodos já implementados.

\item Escreva um procedimento que inverta a ordem dos elementos na Fila. 

\item Implemente uma fila para gerenciar dados do tipo \textbf{Cliente}, onde Cliente é uma ESTRUTURA com Nome e Idade, com todas as operaçőes básicas (FFVazia, Enfileirar, Vazia, Desenfileirar, Tamanho e Imprime). Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Em muitas situaçőes é necessário que exista uma prioridade na Fila, como por exemplo em Bancos onde pessoas idosas tęm prioridade no atendimento. Altere a Fila criada para o tipo {Cliente} para que a inserçăo na Fila seja feita por prioridade de idade, ou seja, se o Cliente que tiver mais de 65 anos deve ter prioridade no atendimento. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\noindent \underline{DICA:} Năo é necessário criar a Fila toda novamente, apenas um novo procedimento de Enfileira por prioridade.

\item Escreva	um	programa	que	simule	o	controle	de	uma	pista	de	decolagem	de	aviőes	em	um	aeroporto.	Neste	programa,	o	usuário	deve	ser	capaz	de	realizar	as	seguintes	tarefas:	

\begin{enumerate}
	\item 	Listar	o	número	de	aviőes	aguardando	na	fila	de	decolagem;	
	\item 	Autorizar	a	decolagem	do	primeiro	aviăo	da	fila;	
	\item 	Adicionar	um	aviăo	ŕ	fila	de	espera;	
	\item 	Listar	todos	os	aviőes	na	fila	de	espera;	
	\item 	Listar	as	características	do	primeiro	aviăo	da	fila.
\end{enumerate}

Considere	que	os	aviőes	possuem	um	nome	e	um	número	inteiro	como	identificador.	Adicione outras	características	conforme	achar	necessário.	

\end{enumerate}
