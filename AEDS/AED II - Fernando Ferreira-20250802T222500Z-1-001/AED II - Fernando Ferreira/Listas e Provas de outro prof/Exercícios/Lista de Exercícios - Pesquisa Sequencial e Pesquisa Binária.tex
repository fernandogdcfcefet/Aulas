\begin{enumerate}

\item Implemente uma TAD Lista para o tipo Aluno (Matrícula e Nome), com as funcionalidades básicas de uma TAD (Insere, Remove, Tamanho, Vazia, FLVazia), e adicione a TAD as seguintes funcionalidades:
\begin{enumerate}
	\item Uma funçăo PesquisaSequencial, que recebe como parâmetro o nome de um aluno e retorna a sua matrícula.
	\item Uma funçăo PesquisaBinaria, que recebe como parâmetro a matrícula de um aluno e retoorne o seu nome.
\end{enumerate}


\item Considere as seguintes estruturas para o Tipo Árvore Binária:

\begin{lstlisting}
struct Aluno{
 char nome[51];
 int matricula;
};

struct No{
 Aluno item;
 No *pai, *esq, *dir;
};

struct ArvoreBinaria{
 No *raiz
 int tam;
};

\end{lstlisting}

Escreva os seguintes algoritmos para esta árvore binária:

\begin{enumerate}
	\item void Inicializa (ArvoreBinaria *A); que inicializa a estrutura da árvore binária.
	\item Aluno Pesquisa (ArvoreBinaria *A, int matricula); que pesquisa um aluno baseado na matricula.
	\item int Tamanho (ArvoreBinaria *A); que retorna o tamanho da árvore binária.
	\item bool Vazia (ArvoreBinaria *A); que verifica se a árvore binária está vazia.
	\item void Insere (ArvoreBinaria *A, Aluno item); que insere um novo item na árvore binária.
	\item int Altura (ArvoreBinaria *A); retorna a altura da árvore binária.
	\item void ImprimirEmOrdem (ArvoreBinaria *A); que imprime os dados da árvore binária em ordem.
	\item void ImprimirPreOrdem (ArvoreBinaria *A); que imprime os dados da árvore binária em pré ordem.
	\item void ImprimirPosOrdem (ArvoreBinaria *A); que imprime os dados da árvore binária em pós ordem.
\end{enumerate}

\item Insira os números abaixo na ordem que săo apresentados numa árvore binária de
busca. Mostre todos os passos.
50; 20; 10; 40; 60; 30; 45; 25; 35; 55

\item A partir da árvore obtida no exercício anterior, remova os números abaixo na ordem
que săo apresentados. Mostre todos os passos.
50; 20; 10; 40; 60; 30; 45; 25; 35; 55

\item Análise a árvore gerada na questăo 3 e responda:

\begin{enumerate}
	\item Qual o custo para se pesquisar o valor 25 na árvore? Explique.
	\item Qual o custo para se pesquisar o valor 25 se usassemos uma estrutura de pesquisa sequencial. Explique.
	\item Qual o custo para se pesquisar o valor 25 se usassemos uma estrutura de pesquisa binária. Explique.
\end{enumerate}


\end{enumerate}
