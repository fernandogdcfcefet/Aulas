	\begin{center}
		{\Large \bf Lista de Exercícios - Listas Lineares} \\
		%{\normalsize \bf em dupla}
	\end{center}
	

\begin{enumerate}
	
\item Implemente uma Lista para gerenciar dados do tipo \textbf{inteiro}. Sua TAD deve conter as seguintes operaçőes:

\begin{enumerate}
	\item Inicializar a Lista (FLVazia)
	\item Verificar se a Lista está vazia (Vazia)
	\item Inserir um novo item na Lista (Insere)
	\item Remover um elemento da Lista (Remove)
	\item Retorna o tamanho da Lista (Tamanho)
	\item Imprime o conteúdo da Lista (Imprime)
	\item Ordenar os dados da Lista (Ordena)
\end{enumerate} 

\noindent Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Implemente um procedimento CLONAR para a Lista, que retorna uma Lista exatamente igual ŕ variável sobre a qual o método foi invocado. Implemente um para a versăo usando arranjos e outro para a versăo usando ponteiros.

\item Implemente uma funçăo que receba como parâmetro duas Listas e retorne uma terceira contendo a concatenaçăo das duas. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Implemente um procedimento MAIOR para a Lista, que retorne o maior elemento presente na lista. Implemente um para a versăo usando arranjos e outro para a versăo usando ponteiros.

\item Implemente um procedimento MENOR para a Lista, que retorne o menor elemento presente na lista. Implemente um para a versăo usando arranjos e outro para a versăo usando ponteiros.

\item Implemente uma Lista para gerenciar dados do tipo \textbf{Pessoa}, onde Pessoa é uma ESTRUTURA com Código e Nome, com todas as operaçőes básicas (FLVazia, Insere, Vazia, Remove, Tamanho e Imprime). Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Escreva uma funçăo ``Pesquisa'' para a Lista do tipo \textbf{Pessoa} que receba como parâmetro o nome de uma Pessoa e retorne seu código caso a Pessoa esteja na lista, caso contrário retorne -1. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Utilizando a Lista do tipo \textbf{Pessoa}, implemente dois procedimentos que sejam capazes de ordenar a lista, um procedimento ordenando pelo código em ordem crescente e outro procedimento ordenando pelo nome em ordem alfabética. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.
	

\end{enumerate}
