\item Alguns editores de texto permitem que algum caractere funcione como um ``cancela-caractere'', cujo efeito é o de cancelar o caractere anterior na linha que está sendo editada. Por exemplo, se `\#' é o cancela-caractere, entăo a sequęncia de caracteres PEM\#\#UB\#C corresponde ŕ sequęncia PUC. Outro comando encontrado em editores de texto é o ``cancela-linha'', cujo efeito é o de cancelar todos os caracteres anteriores na linha que está sendo editada. Neste exemplo vamos considerar `/' como o caractere cancela-linha. Finalmente, outro comando encontrado em editores de texto é o ``salta-linha'', cujo efeito é o de causar a impressăo dos caracteres que pertencem ŕ linha que está sendo editada, iniciando uma nova linha de impressăo a partir do caractere imediatamente seguinte ao caractere salta-linha. Por exemplo, se `@' é o salta-linha, entăo a sequęncia de caracteres ICEI@PUCMINAS.@ corresponde ŕs duas linhas abaixo: 

ICEI

PUCMINAS.

\section*{Filas}

\begin{enumerate}
\item Implemente uma fila para gerenciar dados do tipo \textbf{Cliente}, onde Cliente é uma ESTRUTURA com Nome e Idade. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Implemente um procedimento CLONAR para a Fila, que retorna uma Fila exatamente igual ŕ variável sobre a qual o método foi invocado. Implemente um para a versăo usando arranjos e outro para a versăo usando ponteiros.

\item Implemente uma funçăo que receba como parâmetro duas Filas e retorne uma terceira contendo a concatenaçăo das duas. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Escreva um procedimento que inverta a ordem dos elementos na Fila. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros. \underline{DICA:} considera a utilizaçăo de estruturas auxiliares, como uma pilha.

\item Em muitas situaçőes é necessário que exista uma prioridade na Fila, como por exemplo em Bancos onde pessoas idosas tęm prioridade no atendimento. Altere a Fila criada no exercício 1 para que a inserçăo na Fila seja feita por prioridade de idade, ou seja, se o Cliente que tiver mais de 65 anos deve ter prioridade no atendimento. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\noindent \underline{DICA:} Năo é necessário criar a Fila toda novamente, apenas um novo procedimento de Enfileira por prioridade.

\end{enumerate}

\noindent \underline{DICA:} Use a funçăo \textbf{strcmp}, da biblioteca \textbf{$<$string.h$>$} para realizar a comparaçăo de \textit{strings}. \\

\textbf{strcmp: int strcmp( const char *lhs, const char *rhs );} \\ \\
Essa funçăo compara duas strings segundo sua ordem alfabética e retorna um inteiro.
Se esse inteiro for negativo, é porque a primeira \textit{string} é menor que a segunda.
Se retornar um inteiro positivo, é porque a segunda \textit{string} é maior que a segunda.
Se retornar 0, é porque as strings săo idęnticas.

\item Utilizando a Lista criada no Exercício 1, implemente dois procedimentos que sejam capazes de ordenar a lista, um procedimento ordenando pelo código em ordem crescente e outro procedimento ordenando pelo nome em ordem alfabética. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\noindent \underline{DICA:} Utiliza algum algoritmo de ordenaçăo visto em sala.
