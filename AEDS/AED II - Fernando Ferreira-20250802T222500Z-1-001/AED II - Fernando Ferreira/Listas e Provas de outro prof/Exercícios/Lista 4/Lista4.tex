\section*{Listas}

\begin{enumerate}
\item Implemente uma Lista para gerenciar dados do tipo \textbf{Pessoa}, onde Pessoa é uma ESTRUTURA com Código e Nome. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Implemente um procedimento CLONAR para a Lista, que retorna uma Lista exatamente igual ŕ variável sobre a qual o método foi invocado. Implemente um para a versăo usando arranjos e outro para a versăo usando ponteiros.

\item Implemente uma funçăo que receba como parâmetro duas Listas e retorne uma terceira contendo a concatenaçăo das duas. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\item Escreva uma funçăo ``Pesquisa'' para a lista criada no exercício 1 que receba como parâmetro o nome de uma Pessoa e retorne seu código caso a Pessoa esteja na lista, caso contrário retorne -1. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\noindent \underline{DICA:} Use a funçăo \textbf{strcmp}, da biblioteca \textbf{$<$string.h$>$} para realizar a comparaçăo de \textit{strings}. \\

\textbf{strcmp: int strcmp( const char *lhs, const char *rhs );} \\ \\
Essa funçăo compara duas strings segundo sua ordem alfabética e retorna um inteiro.
Se esse inteiro for negativo, é porque a primeira \textit{string} é menor que a segunda.
Se retornar um inteiro positivo, é porque a segunda \textit{string} é maior que a segunda.
Se retornar 0, é porque as strings săo idęnticas.

\item Utilizando a Lista criada no Exercício 1, implemente dois procedimentos que sejam capazes de ordenar a lista, um procedimento ordenando pelo código em ordem crescente e outro procedimento ordenando pelo nome em ordem alfabética. Implemente uma para a versăo usando arranjos e outra para a versăo usando ponteiros.

\noindent \underline{DICA:} Utiliza algum algoritmo de ordenaçăo visto em sala.

\end{enumerate}
