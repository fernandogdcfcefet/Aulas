\section{Análise de Complexidade}

	\item Suponha dois algoritmos $A$ e $B$, com funçőes de complexidade:
	
	\begin{enumerate}
		\item $A(n) = n^2 - n + 289$ 
		
		\item $B(n) = 7n + 49$
		
		
	\end{enumerate}
	
	Determine, utilizando a notaçăo $O$ o grau de complexidade de cada um dos algoritmos.   
	
	
	\item Determine a complexidade o programa abaixo. O que esse algoritmo faz?
	\begin{lstlisting}
	void main (){
		int i=0, n=0, s=0, V[100];
		scanf(" %d",&n);
		do{
		  scanf("%d",V[i]);

	\end{lstlisting}
	
	
	
	\item O que significa dizer: $O(f(n)) + O(g(n)) = O(max(f(n), g(n)))$ ? \\
	

	\item Determine a (análise de) complexidade de cada um dos algoritmos abaixo e indique sua ordem de complexidade. Considere o melhor e o pior caso para cada um.
	
	\begin{enumerate}
		\item Somar matrizes
		\begin{lstlisting}
		for(i=0; i<n; i++)
			for(j=0; j<n; j++)
				a[i][j] = b[i][j] + c[i][j];
		\end{lstlisting}
	
		
		\item Multiplicar matrizes
		\begin{lstlisting}
		for(i=0; i<n; i++)
			for(j=0; j<n; j++)
			{
				a[i][j] = 0;
				for(k=0; k<n; k++) 
					a[i][j] += b[i][k] * c[k][j];
			}
		\end{lstlisting}
		
	
		
		
		\item Encontrar o maior valor de um vetor
		\begin{lstlisting}
		maior = vet[0];
		for(i=1; i<n; i++)
			if(vet[i]>valor)
				maior = vet[i];
		\end{lstlisting}
		
	
		
		\item Encontrar o maior e o menor valor de um vetor
		\begin{lstlisting}
		maior = vet[0];
		menor = vet[0];
		for(i=1; i<n; i++)
			if(vet[i]>valor)
				maior = vet[i];
			else
				if(vet[i]<menor)
					menor = vet[i];
		\end{lstlisting}
	
		
		
		\item Transpor matrizes
		\begin{lstlisting}
		for(i=0; i<n; i++)
			for(j=i+1; j<n; j++)
			{
				tmp = a[i][j];
				a[i][j] = a[j][i];
				a[j][i] = tmp;
			}
		\end{lstlisting}
	
		
		\item Laço de repetiçăo maluco
		\begin{lstlisting}
		cont = 0;
		for(i=1; i<=n; i*=2)
			for(j=0; j<n; j++)
				cont++;
		\end{lstlisting}
		
	
		
		
		\item Encontrar o k-ésimo valor em um vetor năo ordenado
		\begin{lstlisting}
		int selec_kesimo (int a[], int k, int n)
		{
			int i, j, mini, tmp;
			for(i=0; i<k; i++)

				mini = i;
				for(j=i+1; j<n; j++)

\section{Recursividade}

\begin{enumerate}

\item O máximo divisor comum ($MDC$) de dois números inteiros $x$ e $y$ pode ser calculado 
usando-se uma definiçăo recursiva:  

\begin{equation*}
MDC(x, y) = MDC(x-y, y), ~se ~ x > y
\end{equation*}

Além disso, sabe-se que:

\begin{equation*}
MDC(x, y) = MDC(y, x) 
\end{equation*}


\begin{equation*}
MDC(x, x) = x
\end{equation*}

Exemplo: 
$MDC(10, 6) = MDC(4, 6) = MDC(6, 4) = MDC(2, 4) = MDC(4, 2) = MDC (2,2) = 2$


Crie uma funçăo recursiva para descrever tal definiçăo. Crie, 
também, um algoritmo que leia os dois valores inteiros e utilize a funçăo criada para 
calcular o $MDC$ de $x$ e $y$, e imprima o valor computado.

\item Considere um sistema numérico que năo tenha a operaçăo de adiçăo implementada e 
que vocę disponha somente dos operadores (funçőes) sucessor e predecessor. Entăo, 
pede-se para escrever uma funçăo recursiva que calcule a soma de dois números $x$ e $y$ 
através desses dois operadores: sucessor e predecessor.

\item Implemente uma funçăo recursiva que, dados dois números inteiros $x$ e $n$, calcula o 
valor de $x^n$.

\item Considere uma partida de futebol entre duas equipes $A$ x $B$, cujo placar final é $m$ x 
$n$, em que $m$ e $n$ săo os números de gols marcados por $A$ e $B$, respectivamente. 
Implemente um algoritmo recursivo que imprima todas as possíveis sucessőes de 
gols marcados. Por exemplo, para um resultado de $3$ x $1$ as possíveis sucessőes de 
gols săo ``A A A B", ``A A B A", ``A B A A" e ``B A A A". 

\item Dado um vetor de inteiros $A$, apresente algoritmos recursivos para calcular:

\begin{enumerate}
\item o elemento máximo do vetor
\item o elemento mínimo do vetor
\item a soma dos elementos do vetor
\item o produto dos elementos do vetor
\item a média dos elementos do vetor
\end{enumerate}

\end{enumerate}
