Para Listas Lineares com Arranjo

insere x na posiçăo p deslocando todos os itens com índice >p para a direita

deslocando os itens com a partir da posiçăo removida para as posiçőes anteriores.

Para Listas Lineares com Apontadores

void Retira(TipoApontador p, TipoLista *Lista)
